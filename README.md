# embedded-software-final-proj
Final Project for Embedded Software Engineering (ECEN 2270)

This project will aim to create a playable version of Battleship on the STM32F429-DISC 1 board. 
The first version of the project will only rely on HAL but aspects of RTOS will be attemped to be implemented.
The second version, if time permits, will actually implement FreeRTOS and include a Segger analysis of the 
scheduling and priority.
